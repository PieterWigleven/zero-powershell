# Get-ZNSettingsIdp

## SYNOPSIS
Get identity provider settings in Identity Providers

## SYNTAX

```
Get-ZNSettingsIdp [<CommonParameters>]
```

## DESCRIPTION
Get identity provider settings in Identity Providers

## EXAMPLES

### Example 1: Get Idp settings
```powershell
Get-ZNSettingsIdp

IdentityProviderType IsDefault
-------------------- ---------
azure                False
```

This cmdlet returns the Identity Provider settings.
