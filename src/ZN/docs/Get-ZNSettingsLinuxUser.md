# Get-ZNSettingsLinuxUser

## SYNOPSIS
Get the linux user settings in Asset Managment

## SYNTAX

```
Get-ZNSettingsLinuxUser [<CommonParameters>]
```

## DESCRIPTION
Get the linux user settings in Asset Managment

## EXAMPLES

### Example 1: Get Linux user settings
```powershell
Get-ZNSettingsLinuxUser
```

```output
zn-admin
```

This cmdlet returns the Linux user settings.
