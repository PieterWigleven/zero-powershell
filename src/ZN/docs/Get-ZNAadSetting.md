# Get-ZNAadSetting

## SYNOPSIS
Get Azure AD Settings

## SYNTAX

```
Get-ZNAadSetting [<CommonParameters>]
```

## DESCRIPTION
Get Azure AD Settings

## EXAMPLES

### Example 1: Get AAD Setting
```powershell
Get-ZNAadSetting
```
